HTML

HTML, or Hypertext Markup Language, is a language used to create web pages and web applications. It was first created by Tim Berners-Lee in 1991 as a way to share scientific information with colleagues. HTML has since undergone many revisions and updates, with the latest version being HTML5.

Components

Definition

Tags

 HTML uses tags to identify different parts of a web page. Tags are written in angle brackets (< >) and come in pairs, with the opening tag and closing tag surrounding the content.

Attributes

 HTML tags can also have attributes that provide additional information about the tag. Attributes are written inside the opening tag and typically take the form of name-value pairs.

Elements

 HTML elements are made up of a combination of tags, attributes, and content. They define the structure and content of a web page.

Entities

 HTML entities are special characters that have a specific meaning in HTML, such as the ampersand (&) and the less-than symbol (<). They are used to display special characters that would otherwise be interpreted as HTML code.
