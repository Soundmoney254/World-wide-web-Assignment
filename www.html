HTML

Hypertext Markup Language is a standard markup language for creating webpages and applications. Web server receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. Markup Language describes the structure of the content in a document, providing a way to add meaning to text. Web browsers render HTML documents, as well as other resources such as images and scripts, to create the final webpage displayed to the user.
